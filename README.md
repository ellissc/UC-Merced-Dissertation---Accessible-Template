# Accessibility compliant LaTeX template for UC Merced Thesis/Dissertation

Requires LuaLaTeX

This LaTeX template does the tagging and bookmarking, and takes care of the artifacts in the front matter. 

You'll need to manually add the alt text for your graphs: 
`\includegraphics[width=\linewidth,alt={Descriptive alt text here.}]{image.png}`

You can use [VeraPDF](https://verapdf.org/home/) to check the compatibility.
